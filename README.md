# Additional Features
Every kill chance to get random buff.
- Attack : +20
- Health : +5
- Speed : +1

# How to Play
- Download the Repository
- Extract build.zip
- Open the Folder Build
- Then Run .exe
- Setup prefered resolution size
- ENJOY !!!
